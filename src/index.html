Habilidades

Tecnologias conhecidas:

Html

É uma linguagem baseada em marcação, o que significa que você utiliza tags (etiquetas) para envolver partes específicas do conteúdo, dando-lhes significado e estrutura.

Css

É uma linguagem de estilo utilizada para controlar a apresentação e o layout de documentos HTML.

A principal finalidade do CSS é separar o conteúdo (HTML) da apresentação (estilo), o que permite um maior controle e flexibilidade no design de páginas da web.

Bootstrap

É um dos frameworks de design de código aberto mais populares para o desenvolvimento de interfaces de usuário (UI) responsivas e estilizadas para sites e aplicativos da web.

Javascript

É uma linguagem de programação amplamente usada para desenvolvimento web e é uma das tecnologias fundamentais da internet.

Permite que os desenvolvedores adicionem interatividade, comportamento dinâmico e funcionalidades avançadas às páginas da web.

JQuery

É uma biblioteca JavaScript rápida, concisa e de alto nível que simplifica a interação com documentos HTML.

Foi criada para facilitar o desenvolvimento de páginas web interativas e é amplamente usada na indústria de desenvolvimento web.

Node.js

É uma plataforma de desenvolvimento de aplicativos de código aberto que permite aos desenvolvedores criar aplicativos de servidor e de rede altamente escaláveis usando JavaScript. O Node.js é projetado para ser executado no lado do servidor.

Git e GitHub

Git é um sistema de controle de versão distribuído, amplamente utilizado para rastrear e gerenciar as mudanças no código fonte de um projeto de software.

GitHub é uma plataforma de hospedagem de repositórios Git baseada na web que oferece serviços de colaboração e gerenciamento de código.
